📍 Data Sources

Ontario Hockey League (OHL)

All Erie Otters player statistics are sourced from the official OHL (Ontario Hockey League) website.

📌 ontariohockeyleague.com

Local SQLite Database

This dashboard stores all data in a local SQLite database. No data is sent to external servers.

Location: ./erie_otters.db

Update Frequency

Data can be manually refreshed using the admin panel (available in development mode only).

In production, set up scheduled refresh tasks or use official APIs.

Data Accuracy

This dashboard depends on the accuracy of source data. For official records, always check the OHL website directly.

🗄️ Database Schema

teams

  • id - Primary key
  • name - Team name (e.g., "Erie Otters")
  • city - City name
  • league - League (OHL)
  • code - Team code
  • founded_year - When team was founded

players

  • id - Primary key
  • name - Full player name
  • first_name - First name
  • last_name - Last name
  • team_id - Foreign key to teams
  • position - Player position (C, LW, RW, D, G)
  • jersey_number - Jersey number
  • birth_year - Birth year

player_stats

  • id - Primary key
  • player_id - Foreign key to players
  • season - Season year
  • games_played - Games played
  • goals - Goals scored
  • assists - Assists
  • points - Total points (goals + assists)
  • penalty_minutes - Penalty minutes
  • plus_minus - Plus/minus rating
  • source_url - Where data came from
  • scraped_at - When it was last updated

refresh_logs

  • id - Primary key
  • source_url - Source URL scraped
  • refresh_time - When refresh occurred
  • status - Success or error
  • record_count - Records updated
  • error_message - If there was an error
