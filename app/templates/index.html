📚 About This Dashboard

Learn SQL

Every page shows the actual SQL query used to retrieve the data. This is a great learning resource for understanding how databases work and how to query them.

Erie Otters Data

Comprehensive statistics for the Erie Otters (OHL) covering 2010-2019. Featuring legendary players like Connor McDavid, Bo Horvat, and Alex DeBrincat.

One Database Per Season

This dashboard uses a unique architecture with separate SQLite databases for each season (2010-2019), allowing for independent season analysis and efficient data management.

Local Development

This app runs on your local machine with SQLite. No external servers needed for learning and exploration.
