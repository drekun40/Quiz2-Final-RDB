📚 About Learning Notes

Every page in this explorer displays SQL queries in "Learning Note" sections. These show you the exact SQL code used to retrieve the data you're viewing. This is a great way to learn how databases work!

Click on any "Learning Note" header to expand or collapse the SQL query display. You'll see:

  • What data is being selected
  • Which tables are being used
  • How filters and joins work
  • Real SQL from your database
